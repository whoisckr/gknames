GK Names

This tool is intended to romanize Hangul artist names for Genius Korea when there is no official/preferred romanization from the artist/agency.

FAQ
Is this tool affiliated with Genius?

No. GK Names is a community tool created by ckr for the Genius Korea community. It has no affiliation with ML Genius Holdings, LLC.


Can I see what romanizations are mapped?

Sure, check the Romaja Map! The tool is generally based on romanizations of specifically names. Thus, we haven't established or used a standard system.


How can I add a missing character?

You can submit a character and romanization for review via this form. Please give examples of it being used in a name as that is the scope of the tool.


How can I report an issue?

You can report an issue via this form. Please include all relevant details such as your input and browser.
